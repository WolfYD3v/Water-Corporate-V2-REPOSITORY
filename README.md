# Water Corporate - Godot Project Repository

## What is Water Corporate?
It's a 3D game idle/incremental game made with Godot. With a big point-and-click direction. \
It features simple 3D models, with a constant pixalated look. And controls very point-and-clicky, with a two stories in one.
